# FlyMan

「フライマン」

（概要・テーマ）
この作品は、人生で初めて作ったゲームです。
当時C言語がよく理解できない中で、友達に教えてもらいながら四苦八苦して開発した作品でした。

(開発時期・期間)
HAL東京1年生時の2017年1月～2017年3月(3ヶ月)

（開発ツール）
・開発環境：Visual Studio2010
・言語：C
・HALの学校ライブラリ：CScreen 

（チームメンバー）
プログラマー1人（自分）

（ゲームシステム）
ジャンプしながらゴールを目指すゲームです。
「男たちの楽園」という青いタイルに乗るとゲームクリアーになります。
赤い棘のタイルに乗るとゲームオーバーになります。

(操作方法)
・AとDキーでプレイヤーキャラクターを左右移動させることができます。
・Wキーでプレイヤーキャラクターをジャンプさせることができます。

(起動方法)
「フライマン(ビルド)」というフォルダーを開いてください。
その中にある「フライマン.exe」というアプリケーションをダブルクリックしてください。
起動直後は画面が2重になって動くと思います。その為まず、画面をフルスクリーンにして
一度ジャンプをするか赤いタイルに落ちてください、そうする事でゲームがリセットされます。
